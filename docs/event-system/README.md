## 1. 훌라왕

싱글+멀티 합산 훌라(원턴승리) 횟수가 가장 많은 사람에게 붙는 칭호.

- **데이터**: `users` 테이블에 `hulaWins` 컬럼 추가 (싱글+멀티 합산 누적). 훌라로 게임을 이길 때마다 +1.
- **랭킹 — 훌라 탭 (신규)**: 싱글/멀티 탭 옆에 추가. `hulaWins > 0`인 사람만 목록에 표시 (0회는 제외). 1위는 순위 숫자 자리에 미니 "훌라" 로고(로고체 폰트 축소, -8~-9deg 정도 삐딱하게), 이름과 횟수는 골드색.
- **표시 위치**: 훌라왕인 사람은 **이름이 나오는 모든 곳**에서 이름 텍스트가 골드색으로 보임 (아이콘/테두리 방식은 검토 후 최종적으로 텍스트 골드색으로 확정 — `hula-king-final.html` 참고).

## 2. 👑/💎 전역 1위 뱃지 확장

기존에 게임 화면(상대/내 캐릭터)과 랭킹 싱글/멀티 탭 1위에만 붙어있던 👑(싱글 전역 1위)/💎(멀티 전역 1위) 뱃지를, **이름이 나오는 모든 곳**으로 확장:

- ✅ 이미 있음: 게임 화면 캐릭터 이름, 랭킹 싱글/멀티 탭
- ❌ 추가 필요: 게임 화면 승리 화면(`#win-name`), 정산결과 카드, 메인 화면 프로필칩, 설정 화면 프로필헤더, 대기실 참가자 목록, 이벤트 화면(신규)
- **제외**: 관리자 "유저 관리" 모달 (CRUD 툴이라 랭킹 표시 대상 아님)
- 훌라왕 골드 텍스트도 위와 동일한 범위에 같이 적용
- **주의**: 새로 추가하는 곳마다 줄바꿈 안 되는지 확인 필요 (기존 곳들은 이미 `white-space:nowrap` 패턴으로 안전하게 처리되어 있음, 그 패턴 재사용)

## 3. 주간 이벤트 보상

이번 주(월~일) 기준, 부문별로 서로 다른 보상:

| 부문 | 기준 | 보상 |
|---|---|---|
| 최다판 | 싱글+멀티 합산 플레이 판수 1위 | 싱글 500점만 |
| 최다승 | 싱글+멀티 합산 승수 1위 | 5만원만 |
| 최다 훌라 (주간) | 그 주에 훌라를 한 횟수 1위 (전역 훌라왕 누적과는 별개의 주간 집계) | 500점 + 5만원 둘 다 |

규칙:
- 부문별 독립 판정 — 한 사람이 여러 부문 1위면 해당 보상을 전부 받음 (합산)
- 특정 부문 1위가 0(0판/0승/0회)이면 **그 부문은 그 주에 아무도 안 받음**
- 서버가 매주 자동으로 계산·지급 (관리자 확인 절차 없음). 요일/시각 기준과 중복 지급 방지용 기록 테이블은 구현 시 설계
- 지급은 이미 완료된 상태로, 유저가 지급 이후 **처음으로 메인 화면에 들어왔을 때** 축하 레이어 1회 표시 (여러 부문 받았으면 합계까지 같이 표시) 후 다시 안 뜸

## 4. 이벤트 메뉴 (신규 화면) — 보류 중

**사용자가 "일단 킵해두자"고 명시적으로 보류 지시함. 다른 항목보다 우선순위 낮음.**

- 메인 화면 하단에 "이벤트" 아이콘 추가 (랭킹/설정/도움말과 같은 자리, 4분할, 골드톤 배경 + 빨간 점으로 진행 중 표시)
- 카운트다운 배너 (다음 정산까지 남은 시간), 이번 주 기간 표시
- 최다판/최다승/최다훌라 실시간 순위 (각 3위까지, LIVE 태그, 카드마다 "1위 보상" 안내문구)
- 지난 주 수상자 카드 (부문별 실제 지급액 표시)
- "전체보기" 버튼 → 역대 수상자 화면 (주차별 최신순 나열, 특정 부문에 아무도 없었던 주는 회색으로 "이번 주는 없음" 명시)

## 5. 깜짝 2배 이벤트 판

- **트리거**: 게임 생성 시점마다 낮은 확률(예: 10%, 정확한 %는 미정 — 반응 보고 조절)로 그 판을 "2배 판"으로 랜덤 지정. 매판 독립적으로 굴림 (직전 결과와 무관). 싱글/멀티 둘 다 적용.
- **효과 범위**: **승자가 받는 정산 금액만 2배** (패자 벌금은 그대로 — 부담 없이 다들 신나게 참여하도록 하는 의도적 설계)
- **중요**: 승패 판정이나 승수/승률 집계에는 전혀 영향 없음. 오직 정산 `pointChange` 금액에만 곱해짐 (`singleWins`/`multiWins`/게임 수 집계 로직은 그대로).
- **연출**: 카드 배분 애니메이션 직후, 훌라 스탬프와 같은 방식(광선+텍스트 팝인, `.hula-stamp` 애니메이션 재사용)으로 "⚡2배 이벤트⚡" 전체화면 오버레이가 3초간 재생. **3초 후엔 완전히 사라짐 — 상단에 계속 남는 배지 없음** (처음엔 상시 배지 아이디어였다가 명시적으로 뺌).
- **오버레이는 실제 게임 화면 위에 얹는 것뿐이라, 화면 방향/크기와 무관하게 항상 동일하게 잘 뜸** (게임 화면 자체의 가로모드 레이아웃은 안 건드림).
- **타이밍**: 이 오버레이가 다 끝나야 실제 턴 타이머가 시작되고 조작 가능해짐 — 기존 카드 배분 애니메이션과 동일한 방식으로, 오버레이 재생 중에 시간이 깎이지 않게 함.
- **정산결과 화면**: 상단에 안내 배너("⚡ 이번 판은 2배 이벤트였어요!") + 승자 카드에만 "⚡2배 적용" 태그. 이미 2배 계산된 최종 금액만 표시 (계산식은 노출 안 함).
