게임 화면 레이아웃 자체는 그대로 두고(안 건드리기로 하셨으니까), 그 위에 "잠깐 뜨는 오버레이"만 얹는 방식이에요 — 훌라(원턴승리) 연출이랑 같은 방식으로, 기존 화면을 전혀 재구성하지 않아요.

화면 전체를 살짝 어둡게 깔고 "⚡2배 이벤트⚡"가 훌라 스탬프랑 똑같은 방식(광선+팝인)으로 터짐. 3초 뒤엔 아무것도 안 남고 완전히 사라져요 — 상단에 계속 떠 있는 배지 없앴어요.

B. 정산결과 화면
기존 정산 카드(`.pcard`) 스타일 그대로, 상단에 안내 배너 하나 추가하고 승자 카드에 "⚡2배 적용" 태그 + 실제 2배 반영된 금액 표시.

  • 승자 카드에만 "⚡2배 적용" 태그를 붙였어요 — 2배는 승자 보상에만 적용되니까, 패자 카드엔 태그 없이 평소 금액 그대로.
  • 상단 안내 배너는 이 판이 2배였다는 걸 다시 한번 상기시켜주는 용도 — 태그만 보면 놓칠 수 있어서 같이 뒀어요.
  • 실제 금액(+2,400원)은 이미 2배 계산이 끝난 최종값이에요 — 별도로 "1,200원 × 2"처럼 계산식을 보여주진 않고, 태그로 "2배였다"는 것만 알려줘요.
