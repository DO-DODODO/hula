  • 진행 중인 순위엔 "LIVE" 태그를 붙여서, 지난 주 결과랑 헷갈리지 않게 구분했어요.
  • 지난 주 수상자 카드는 예시처럼 같은 사람이 둘 다 받으면 두 줄 다 나와요(따로 합계 표시는 안 하고 각 항목 그대로 나열 — 메인화면 축하 레이어에서 합계는 이미 보여줬으니까 여기선 내역 위주).
  • 카운트다운 배너는 항상 상단 고정 — 마감 임박하면 색을 좀 더 다급하게(빨간기) 바꾸는 것도 나중에 고려 가능.
  • 보상이 부문마다 달라요 — 최다판 500pt만, 최다승 5만원만, 최다 훌라(이번 주 기준)는 500pt+5만원 둘 다. 그래서 각 카드 상단에 "1위 보상" 문구를 작게 넣어서 뭘 걸고 경쟁하는지 바로 보이게 했어요. 최다 훌라 카드는 골드 톤으로 살짝 더 특별해 보이게.
  • 당장은 세 카드지만, 나중에 다른 이벤트 생기면 이 아래로 카드만 추가하면 되는 구조예요.
